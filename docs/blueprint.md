# **App Name**: TechZone E-Store

## Core Features:

- Product Catalog: Browse and view electronic components with detailed specifications.
- Shopping Cart: Add components to a virtual shopping cart for purchase.
- Order Management: Process orders and manage inventory.
- User Authentication: Secure user accounts with login and registration.
- Admin Dashboard: Manage products, inventory, orders, and user accounts.
- AI Component Suggestion Tool: AI tool provides intelligent component suggestions based on project requirements. This LLM-powered tool uses reasoning to consider many parameters when making the suggestions.
- Inventory Alerts: Receive notifications for low stock levels of electronic components.

## Style Guidelines:

- Primary color: Deep blue (#0D47A1) to convey technology and reliability.
- Background color: Light gray (#F5F5F5) for a clean, neutral backdrop.
- Accent color: Light cyan (#00BCD4) to highlight interactive elements.
- Headline font: 'Poppins' (sans-serif) for a modern, clean feel. Note: currently only Google Fonts are supported.
- Body font: 'Roboto' (sans-serif) for readability. Note: currently only Google Fonts are supported.
- Use Font Awesome or Boxicons for consistent and recognizable icons.
- Implement a Header + Sidebar + Content layout for the admin panel and a Header + Category + Product grid layout for the user interface.
- Apply subtle shadows and corner rounding to product cards for visual appeal.
